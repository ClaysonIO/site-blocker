Site Blocker Chrome Extension

About

Site Blocker is a Chrome extension that helps you stay focused by blocking access to websites that you find distracting. Simply add domains to your personal block list, and the extension will prevent you from navigating to those sites.

Key Features

  • Block specific domains from being accessed
  • Set optional time ranges for when sites should be blocked
  • Enable/disable blocking for specific sites
  • Redirect to productive alternatives instead of showing a block page
  • User-friendly interface for managing blocked sites
  • Customizable blocking page with visual feedback

Installation (Development Mode)

  1. Clone or download the repository from GitHub
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" using the toggle in the top-right corner
  4. Click "Load unpacked" and select the directory containing this extension
  5. Generate the icon files using the included icon generator
